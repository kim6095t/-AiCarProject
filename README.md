# AikitCarProjectAi

리액트 & 노드서버 & esp32-cam을 이용한 프로젝트입니다

1. 노드서버 라즈베리파이4에 구동
2. 포트포워딩을 통한 연결
3. 웹에서 서버에 접속하여 실행
(웹이 실행된 이후에는 esp32-cam과 direct wifi 사용하여 영상을 받아오기)
